Tic-Stack-Toe is a Tic-Tac-Toe styled game modeled off of Gobblet

This application uses

  • ActionBarSherlock library (released under the Apache 2.0 license) in order to provide version 4 Android Functionality on pre-v4 devices.
  • ActionBar- Pull to Refresh (released under the Apache 2.0 license).
  • Smooth Progress Bar library (released under the Apache 2.0 license) used by the ActionBar-PullToRefresh library.
  • Facebook-like-button
  • UndoBar to allow undoing match dismissals.
